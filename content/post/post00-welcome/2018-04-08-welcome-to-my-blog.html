Welcome to my new blog, which I’m using as an opportunity to explore new data analysis tools. This is intended as an exercise in learning-by-doing - as I learn new tools, I will be using them to explore real-life data sets.

Most of the analysis carried out in this blog will use the R programming language. I’m increasingly relying on this powerful and versatile tool to complement Microsoft Excel in my day to day analysis For anyone getting started in R, I highly recommend the excellent and accessible R for Data Science by Hadley Wickham and Garrett Grolemund.

This website is itself written using R. Having learned the basics of R, including how to write R Markdown files (which allow you to blend your code, output and commentary into a single document), you can publish these as webpages. This is done using the Blogdown package for R. Technical instructions are provided in blogdown: Creating Websites with R Markdown, but beginners such as myself might prefer to start with step-by-step guides provided in other blogs, such as here and here.
